# 仓颉.skill (Cangjie)
> *「不仅看透灵魂，还要穿上皮囊，感受体温」*

**仓颉是「女娲」与「同事」的深度融合。它负责蒸馏任何团队、公众号或机构的写作风格，并将其转化为可运行的高仿生写作引擎。v2.0 新增「情感层」建模和「黄金语料锚定」，目标仿生度 ≥ 95%。**

[女娲.skill](https://github.com/alchaincyf/nuwa-skill) 证明了提炼一个人的认知操作系统是可行的。
但对于内容创作团队，仅仅有逻辑是不够的。你需要他们的笔法、他们的节奏、甚至是他们字里行间流露出的「忧患」或「狂热」。

**仓颉通过 9 路并行 Agent，为您捏造出最像真人的媒体大脑。**

---

## 🌪️ 核心架构：理-文-情-神

输入一个团队名，仓颉执行四位一体提炼：

1. **理 (Logic)**：执行三重验证，提炼核心心智模型与决策启发式（女娲模式）。
2. **文 (Expression)**：量化表达指纹，还原词汇、节奏与排版美学（同事模式）。
3. **情 (Emotion)** ⭐**v2.0 新增**：建模情感效价 (Valence) 与唤醒度 (Arousal)，捕捉共情模式。
4. **神 (Evolution)**：通过 `Correction.md` 补丁，基于用户反馈实现风格的动态对齐。

---

## 🚀 核心优势

- **9 路并行地毯式调研**：不仅扫描著作和访谈，更将微信公众号列为最高权重的白名单水源。
- **量化风格指纹**：拒绝模糊的「风格冷峻」，通过 `Writing_Pattern.json` 硬编码句长、标点密度等参数。
- **黄金语料锚定**：精选 3-5 段原文作为 few-shot 示例直接注入，确保复刻精度突破 95%。
- **自动化风格审计**：内置审计 Agent 对生成文本进行 Stylometric 评估，不达标自动重写。

## 📊 升级对比

| 维度 | 女娲 (Nuwa) | 仓颉 (Cangjie) v2.0 |
|:---|:---|:---|
