# 仓颉.skill v3.0 (Cangjie)
> *「长出大脑，持久进化，复利增长」*

**仓颉 v3.0 是高仿生写作的终极形态。它不再满足于模拟「语气」，而是通过 Karpathy 的「增量 Wiki」模式和即梦实验室的「知识图谱」逻辑，为 AI 打造一个与目标团队完全同步的「持久化大脑」。**

---

## 💎 v3.0 革命性升级
1. **意 (Intent/Wiki) 层建模**：建立持久化的实体知识库（Entities），记录团队对万物的全量历史看法，实现「思想的位移」追踪。
2. **隐喻与类比仓库 (Metaphor Vault)**：结构化采集并调用目标团队的专属修辞武器，让 AI 学会「打比方」。
3. **增量记账 (Bookkeeping)**：每扫描一篇新文章，知识库自动增量更新，产生认知的复利效应。
4. **红蓝对抗审计**：通过独立的审计 Agent 评估「观点一致性」与「仿生指数」，目标仿生度 ≥ 99%。
